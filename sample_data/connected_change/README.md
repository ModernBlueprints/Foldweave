# Connected Change release fixtures

All project content below is synthetic and was generated for Foldweave. No
third-party project content or personal information is included. Historical
internal paths and schema names may retain their original Name Atlas spelling
where strict compatibility requires it.

- `sofia_apollo/` and `martin_apollo/` contain the same 24 logical files in
  different local layouts. Corresponding files have identical payload bytes,
  except that the five Markdown documents use layout-specific supported link
  destinations while preserving all other bytes and logical relationships.
- `ambiguity/` contains a four-file textual case that intentionally does not say
  which of two presentation candidates the user approved.
- Raster, audio, PDF, spreadsheet, and opaque examples are tiny synthetic files.
  They demonstrate byte-preserving carriage, not semantic understanding of
  those formats.
- The production materializer creates `working/templates/empty` in both hero
  copies. Empty directories cannot be represented directly by Git without
  adding a file, which would violate the exact 24-file contract.
